# ChefDelivery 🍔

O **ChefDelivery** é um aplicativo de entrega de refeições desenvolvido durante o curso **"iOS com SwiftUI: construindo componentes e layouts"** da [Alura](https://www.alura.com.br/). O projeto foca na criação de interfaces modernas e declarativas utilizando o framework **SwiftUI**, explorando a construção de componentes personalizados, layouts complexos e a organização de dados através de mocks.

---

## 📱 O Projeto

O aplicativo simula uma plataforma de delivery onde o usuário pode navegar por diferentes categorias de estabelecimentos, visualizar promoções em um carrossel dinâmico e explorar uma lista de lojas disponíveis.

### Principais Funcionalidades:
- **Barra de Navegação Personalizada:** Implementação de uma `NavigationBar` com botões e ícones.
- **Grid de Categorias:** Exibição de tipos de estabelecimentos (Restaurantes, Farmácias, Pet, etc.) utilizando `LazyVGrid`.
- **Carrossel de Promoções:** Banner rotativo implementado com `TabView` para destacar ofertas especiais.
- **Lista de Lojas:** Seção dedicada à listagem de estabelecimentos parceiros com imagens e nomes.
- **Arquitetura Declarativa:** Uso intensivo de `HStack`, `VStack` e `ZStack` para composição de telas.

---

## 🛠️ Tecnologias e Conceitos Utilizados

O desenvolvimento do ChefDelivery permitiu o domínio de diversas ferramentas e conceitos do ecossistema Apple:

| Tecnologia/Conceito | Descrição |
| :--- | :--- |
| **SwiftUI** | Framework declarativo para construção de interfaces. |
| **Stacks (H/V/Z)** | Organização e empilhamento de elementos na tela. |
| **LazyVGrid** | Criação de layouts em grade eficientes para grandes volumes de dados. |
| **TabView** | Implementação de navegação por abas e carrosséis de imagens. |
| **Modificadores** | Personalização de componentes (fontes, cores, espaçamentos, bordas). |
| **Mocks & Models** | Estruturação de dados fictícios para popular a interface de forma dinâmica. |
